  • Desenvolvimento Back-End

    PrograMaria

    Neste curso você vai fazer o seu primeiro servidor usando a linguagem JavaScript e ferramentas como VS Code e Node.JS. Além disso, vai conhecer e usar os gerenciadores de pacotes (como o express). Vai também aprender os métodos HTTP (GET, POST, PATCH e DELETE) e vai realizar a sua primeira API. Vai conhecer mais sobre Banco de Dados e como realizar as quatro operações básicas performadas: o famoso CRUD. Além disso, vai saber mais sobre versionamento, como fazer um deploy, como proteger melhor os dados e como integrar o Back-End com o Front-End! E para isso vai usar algumas ferramentas como o Git, GitHub, Insomnia, MongoDB e o Render.
